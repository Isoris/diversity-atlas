Methods note
defs

π (node label) — within-group nucleotide diversity from the per-group folded SFS.

FST — between-group differentiation. Estimator recorded in the payload (Weir & Cockerham, Hudson, or Reynolds — pipeline-side choice).

DXY — absolute divergence between groups (per-site probability of nucleotide difference).

dA — net divergence = DXY − (π_i + π_j) / 2. Removes the within-group diversity contribution from DXY.

Edge filter — caret menu in the panel header: Show all / Top N (default N = 8, configurable) / Significant only (bootstrap CI excludes 0).

Layout — default circular: clockwise from 12 o'clock, alphabetical / group-order. Toggle to force-directed uses a fixed RNG seed so the layout is reproducible across reloads; edges weighted by 1 − FST so high-divergence pairs repel.
